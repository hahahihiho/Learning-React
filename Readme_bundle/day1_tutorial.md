# How to start React

## initialization
1. download npm
 - node --version
 - npm --version

2. download yarn
 - npm install yarn

3. make inital react app
 - yarn create-react-app [app name]

4. how to run the app
 - yarn start

## How to install package(plug in) by yarn
5. yarn add [plug_in_name]
 - yarn remove [package-name]
6. package.json 에 있는 package 한번에 다운받기
 - yarn install
